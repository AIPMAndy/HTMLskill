# HTMLskill Examples

This directory contains example scripts demonstrating HTMLskill's capabilities.

## Quick Start Examples

### 00-minimal.py
The absolute minimal example - just 3 lines of code to generate a page.

```bash
python3 00-minimal.py
```

### 01-landing-page.py
Simple landing page with basic components.

### 02-complete-landing.py
More comprehensive landing page with multiple sections.

### 03-render-to-html.py
Demonstrates how to render context to HTML output.

### 04-complete-with-output.py
Complete landing page that generates actual HTML file.

```bash
python3 04-complete-with-output.py
# Opens output/complete-landing.html
```

### 05-product-landing.py
Professional SaaS product landing page example.

```bash
python3 05-product-landing.py
# Opens output/product-landing.html
```

## Running Examples

All examples can be run directly:

```bash
# Run any example
python3 examples/00-minimal.py

# Output files are saved to examples/output/
open examples/output/minimal.html
```

## Example Categories

- **Minimal**: Simplest possible examples to get started
- **Landing Pages**: Complete landing page templates
- **Components**: Individual component demonstrations
- **Layouts**: Grid and layout patterns
- **Advanced**: Complex compositions and patterns

## Learning Path

1. Start with `00-minimal.py` to understand the basics
2. Explore `04-complete-with-output.py` for a full example
3. Check `05-product-landing.py` for real-world patterns
4. Experiment with your own variations!
