# HTMLskill Roadmap

## Vision

Make HTMLskill the most developer-friendly Python HTML generation framework, combining simplicity with professional design systems.

## Principles

1. **Python-First**: Pure Python API, no template languages
2. **Design by Default**: Professional output without design expertise
3. **Zero Configuration**: No build tools, instant results
4. **Type Safe**: Full IDE support with type hints
5. **Extensible**: Easy to add custom components
